🔧 Tech Stack:
React.js V-19 | JavaScript (ES6+) | useState, useEffect | Local Storage | Static JSON API (GitHub hosted) | CSS (Light/Dark Theme)

✨ Key Features:
🔐 Role-based Login (Employee / HR) | 📦 API-based data fetching (101 users dataset) | 💾 Persistent login using Local Storage | 👨‍💼 Employee Dashboard (profile + hierarchy view) | 🧑‍💻 HR Dashboard (all employees + search functionality) | 🔍 Search employees by name | 🌗 Light/Dark mode toggle | 🚪 Logout with session reset | 📢 Conditional welcome popup (first-time users)

🔄 Workflow:
User selects role (Employee / HR) | Credentials validated via API | Session stored in Local Storage | Dashboard rendered based on role | Logout clears session

📌 Highlights:
Implemented role-based conditional rendering | Managed state and lifecycle using React Hooks | Designed interactive UI with popups and dynamic data | Simulated real-world authentication flow (frontend-based)

This project uses mock authentication (frontend only). In production, secure backend authentication is recommended.
